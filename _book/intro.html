2 Introduction

This is my personal collection statistical notes and citations which serves as a reference I can use when helping natural scientists do their statistics in R.

This is not intended to be a Statistical Textbook. There are lots of those, many freely available online. What I have found a lack of, is clear instructions for how to apply and, almost more importantly, interpret statistics in R. I have also found a lack of a list of simple best practices.

Consider this a Practical Guide to performing statistics in R for Natural Scientists. In places I do include some statistical guidance, but generally speaking I assume some background in statistics. This is not a source for understanding the underlying theory of statistics!

Everywhere possible I cite my information be it a stackoverflow question or an academic journal article. If you have a citation you think I should include or think there’s something I’m missing, please open an issue in the GitHub repository and let me know (ideally with a citation!)

This is a work in progress!
